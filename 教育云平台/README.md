> ECP(education cloud platform) 教育云平台的静态页

### 项目介绍
  - 以服务国家教育均衡战略、实现教育资源共享为目标，为用户构建大数据教育综合服务云平台 —— 让教、学、管、评更容易。
  
### 项目架构
  - font-awesome(字体) + zui(ui插件) + swiper(轮播插件) + jquery

### 运行环境
  - chrome浏览器pc端
